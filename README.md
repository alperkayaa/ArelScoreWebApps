# ArelScoreWebApp

Merhaba,

Ben Alper Kaya. ArelScore, okul projem kapsamında MVC mimarisi kullanılarak geliştirdiğim bir proje puanlama sistemidir.

Sistem yalnızca okulumdaki öğrenciler tarafından kullanılabilmektedir. Puanlama işlemi ise sadece okula ait .edu uzantılı e-posta adresine sahip kullanıcılar ve yetkilendirilmiş okul hesapları tarafından yapılabilmektedir. Bu sayede güvenli ve kontrollü bir oylama süreci sağlanmıştır.

Projede HTML, CSS, Bootstrap ve JavaScript teknolojilerini kullandım. Backend tarafında API yapısı ile dinamik veri akışı sağladım ve verilerin saklanması ile yönetilmesi için SQL Server veritabanı kullandım.

Sistem üzerinden öğrenciler proje yükleyebilmekte, projeleri oylayabilmekte ve verilen puanlara göre otomatik sıralama yapılmaktadır. En yüksek puanı alan projeler üst sıralarda listelenmektedir.

Bu proje, MVC mimarisini uygulamalı olarak geliştirmek, veritabanı yönetimi, yetkilendirme sistemi ve API entegrasyonu konularında deneyim kazanmak amacıyla hazırlanmıştır.


Hello,

My name is Alper Kaya. ArelScore is a project evaluation system that I developed as part of my school project using the MVC architecture.

The system can only be accessed by students from my university. The voting process is restricted to users who have an official .edu email address belonging to the university and authorized school accounts. This ensures a secure and controlled evaluation process.

In this project, I used HTML, CSS, Bootstrap, and JavaScript for the front-end. On the backend side, I implemented an API structure to manage dynamic data flow and used SQL Server as the database system for storing and managing data.

Within the system, students can upload their projects, vote on other projects, and view automatic rankings based on the given scores. Projects with the highest scores are displayed at the top of the ranking list.

This project was developed to gain practical experience with MVC architecture, database management, authentication/authorization systems, and API integration.
